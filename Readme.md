# Selenium Java Project

## Setup
- java jdk 15+
- maven
- install browsers
  - chrome
  - firefox
  - safari
  - edge

## How to run

```shell
$ mvn clean test
```
 Generate Allure Html report with screenshot
 ```shell
$mvn allure:report
```
